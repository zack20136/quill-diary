# 文件目錄

QuillLockDiary 的設計與流程文件入口。

## 用語對照

面向使用者的文件用語對齊 [`settings_copy.dart`](../lib/features/settings/settings_copy.dart) 頂部註解；程式識別符（型別、Provider、路徑）保留英文。

| 用語 | 說明 |
|------|------|
| 復原金鑰 | 使用者保存的備用金鑰；程式中對應 `Recovery Key`、`recoveryWrapKey` |
| 本機受信任裝置 | 經 Keystore 自動解鎖的授權狀態；程式中對應 `trusted device`、`trustedDevice` |
| 日記庫 | 加密資料本體；實體目錄為 `vault/` |
| 索引資料庫 | 解鎖期間的 SQLite 搜尋索引；含首頁搜尋與標籤顏色快取，路徑見 [索引資料庫.md](./索引資料庫.md) |

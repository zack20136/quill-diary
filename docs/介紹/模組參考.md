# 模組參考

這份文件是 Quill Diary 的模組速查表。

它只回答兩件事：

- 這個型別或模組是什麼
- 它主要負責什麼

流程、行為細節與資料生命週期，請回到對應主題文件。

## 主要型別

| 型別 | 角色 |
|------|------|
| `UnlockedVaultSession` | 已解鎖日記庫的 session，包含 `vaultId`、`trustedDevice`、`recoveryWrapKey`、`deviceSlotId` |
| `RecoveryMetadata` | `vault/recovery.json` 的明文 metadata，記錄 vault id、建立時間、hint 與 KDF 參數 |
| `PendingAttachment` | 尚未正式寫入日記庫的附件 |
| `EditorDraftSnapshot` | 編輯器當前欄位快照，用來判斷是否有未儲存變更 |
| `EditorDraftRecord` | 寫入 `drafts/` 的草稿資料模型 |
| `PersonalizationPreferences` | 個人化偏好聚合模型，包含語言、主題、自動鎖定、圖片品質與排版 |
| `SessionLockReason` | 進入 `locked` 的原因：`inactivity`、`manual`、`authFailed` |
| `PortableImportResult` | 可攜式匯入結果摘要，包含匯入、略過與失敗資訊 |

## 核心模組

| 模組 | 職責 |
|------|------|
| `VaultRepository` | 日記庫初始化、可信裝置 session、復原金鑰解鎖、entry / asset 讀寫、manifest 驗證與索引同步 |
| `DeviceKeyManager` | Android Keystore 互動、可信裝置 key 管理，以及 recovery wrapping key 的包裝與解開 |
| `AppLockService` | 保存 `AppUnlockMode`，並查詢螢幕鎖與生物驗證是否可用 |
| `AppSessionController` | 管理執行期 session、可信裝置解鎖與背景逾時鎖定 |
| `UserPreferences` | 將 vault 外的 `app_preferences.json` 讀寫成個人化偏好 |
| `SessionInactivityWatchdog` | 管理背景逾時計時與 `resumed` 後的短暫前景穩定期 |
| `SessionLifecycleBinding` | 將 App lifecycle / pointer 事件接到 session 鎖定與自動 reauth |
| `unlock_mode_policy.dart` | 解鎖模式能力檢查、可信狀態一致性比對與鎖定提示文案 |
| `IndexDatabaseManager` | 依 session 開啟索引，並管理關閉、刪除與重建 |
| `EditorDraftStore` | 管理 `drafts/{draftKey}/` 下的加密草稿與待上傳附件 |
| `TagStylesStore` | 管理 `vault/tag_styles.json` 的標籤樣式與 accent 顏色資料 |
| `VaultArchiveIo` | 統一完整備份、還原與可攜式匯入匯出的底層 I/O |
| `VaultTransferService` | 串接設定頁的備份、還原、匯入匯出與外部資料夾交付流程 |

## `storage/` 子模組

| 模組 | 職責 |
|------|------|
| `VaultBackupPolicy` | 管理 `backup_*.zip`、`markdown_*.zip`、`html_*.html` 的命名與辨識規則 |
| `VaultBackupIo` | 建立、檢查、預覽與還原 `backup_*.zip` |
| `PortableExportIo` | 產生 Markdown zip 與 HTML 匯出檔 |
| `PortableImportIo` | 匯入資料夾或 zip 內的 `.md`、`.html`、`.htm` 文件 |
| `EasyDiaryBackupImporter` | 辨識並匯入 Easy Diary 完整備份 zip |
| `OpenedZipArchive` / `archive_extract` | 處理 zip 開啟、選讀、解壓與一般檔案複製 |
| `ExternalDirectoryStore` | 記住最近一次使用的外部資料夾 |
| `ExternalDirectoryPicker` | 提供跨平台資料夾選擇；Android 寫入走 SAF |
| `deliverToExternalDirectory` | 將暫存輸出檔交付到外部資料夾 |
| `AndroidSafFileCopy` | 負責 Android SAF 串流寫入 |

## Provider 速查

| Provider / 函式 | 職責 |
|------------------|------|
| `appSessionProvider` | 執行期 session 狀態 |
| `appStartupProvider` | 啟動階段的一次性初始化 |
| `effectiveAppSessionProvider` | 啟動狀態與執行期 session 的合併視圖 |
| `activeVaultSessionProvider` | 只在 `unlocked` 狀態暴露有效 session |
| `personalizationPreferencesProvider` | 個人化偏好設定（語言、主題、自動鎖定、圖片品質、排版） |
| `watchPersonalizationPreferences()` | 在 widget 層讀取已載入的個人化偏好，失敗或載入中時回傳預設值 |
| `readSessionBackgroundTimeout()` | 依個人化偏好讀取目前自動鎖定的背景逾時長度 |
| `unlockModeProvider` | 讀取目前 `AppUnlockMode` |
| `applyUnlockModeChange()` | 設定頁切換解鎖模式，實作見 [`unlock_mode_change.dart`](../../lib/features/settings/unlock_mode_change.dart) |
| `shared/providers/core_providers.dart` | app-wide 基礎依賴的集中組裝點 |
| `editorDraftKeysProvider` | 提供目前仍有本地草稿的 entry key 集合 |

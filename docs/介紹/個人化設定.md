# 個人化設定

這份文件整理 Quill Diary 目前可調整的個人化設定，包括語言、主題、自動鎖定、圖片品質與日記排版。

它聚焦的是「使用者可以改什麼」與「這些設定會影響哪裡」，不展開 session 或加密細節。

## 目前有哪些設定

| 類別 | 可調內容 | 目前狀態 |
|------|----------|----------|
| 語言 | 繁體中文、English | English 顯示為即將推出，尚未啟用 |
| 主題顏色 | 跟隨系統、淺色、深色 | 已實作 |
| 自動鎖定 | `1 / 3 / 5 / 10 分鐘` | 已實作 |
| 圖片品質 | 原圖、標準、高畫質 | 已實作 |
| 日記排版 | 標題字體大小、標題行距、內文字體大小、內文行距、內文段落間距 | 已實作 |

## 各設定會影響什麼

### 語言

- 目前預設為繁體中文
- `English` 已出現在個人化頁，但尚未開放切換
- 等正式啟用後，會影響 App 介面顯示語系

### 主題顏色

- 可切換為 `跟隨系統`、`淺色`、`深色`
- 會直接影響整個 App 的 Material 主題
- 設定在 App 啟動時就會套用

### 自動鎖定

- 選項為 `1 / 3 / 5 / 10 分鐘`
- 單位在頁面上共用一次顯示，不在每個選項重複寫 `分鐘`
- App 進入背景超過所選時間後，再次回到前景時需要重新解鎖
- 預設為 `3 分鐘`

補充：

- 短時間切換 App 不會立刻鎖定
- 備份、還原、匯入匯出等敏感長時間操作進行中，會暫時避免背景逾時鎖定

更完整的鎖定與重新驗證流程，請看 [解鎖與會話.md](./解鎖與會話.md)。

### 圖片品質

這個設定只影響**從相簿新選取、準備插入日記的圖片**。

| 選項 | 行為 |
|------|------|
| 原圖 | 不壓縮，保留原始解析度與檔案大小 |
| 標準 | 長邊縮至 `1280 px`、JPEG 品質 `70` |
| 高畫質 | 長邊縮至 `1920 px`、JPEG 品質 `85` |

這代表：

- 它影響的是新加入編輯器的圖片
- 不會回頭重壓既有附件
- 會影響之後正式儲存進 `vault/` 的圖片大小與清晰度

### 日記排版

可調整：

- 標題字體大小
- 標題行距
- 內文字體大小
- 內文行距
- 內文段落間距

這些設定會影響：

- 編輯器中的閱讀體驗
- 預覽模式的排版
- 主畫面日記列表與摘要文字的呈現

個人化頁也提供「還原預設」功能，可一次恢復預設排版。

## 設定資料放在哪裡

個人化設定不放在 `vault/` 裡，而是單獨存放於 vault 外的偏好檔：

- `app_preferences.json`

這些設定屬於 App 使用偏好，而不是正式日記資料本體。

高層來看：

- `vault/` 保存正式日記、附件與復原設定
- `drafts/` 保存編輯中的暫存草稿
- `app_preferences.json` 保存主題、語言、自動鎖定、圖片品質與排版偏好

## 和其他模組的關係

| 設定 | 主要影響模組 |
|------|--------------|
| 語言 / 主題 | `app/`、整體 UI |
| 自動鎖定 | Session / 解鎖流程 |
| 圖片品質 | 日記編輯器圖片選取與壓縮 |
| 日記排版 | 編輯器、預覽、首頁列表與摘要 |

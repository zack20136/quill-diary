# QuillLockDiary 文件索引

這份文件是 `docs/` 目錄的導覽頁。

- 根目錄的 `README.md`：給第一次進 repo 的人看，偏專案總覽
- `docs/文件索引.md`：給已經在 repo 內的人快速找文件，偏文件導航

如果只想先看目前專案的核心流程，建議順序是：

1. [`加密流程.md`](./加密流程.md)
2. [`解密流程.md`](./解密流程.md)
3. [`其他主要流程.md`](./其他主要流程.md)

## 流程文件

| 文件 | 內容 |
|------|------|
| [**加密流程.md**](./加密流程.md) | 從明文到 `.enc`：`fileKey`、`LDJ2` 檔案版面、AES-GCM content、雙槽包裝 |
| [**解密流程.md**](./解密流程.md) | 受信任裝置如何 unwrap、Recovery Key 如何導出 wrapping key、驗證 Manifest |
| [**其他主要流程.md**](./其他主要流程.md) | 建立 Recovery、開啟 session、rewrap、索引重建、備份／還原／匯出 |

## 對應程式碼

- `lib/infrastructure/crypto/crypto_service.dart`：`LDJ2` 格式、encrypt / decrypt、Recovery KDF
- `lib/infrastructure/storage/vault_repository.dart`：session、存取、備份／還原／匯出、rewrap
- `lib/infrastructure/security/device_key_manager.dart`：Android Keystore 與 secure storage 橋接
- `lib/infrastructure/markdown/front_matter_codec.dart`：front matter encode / decode
- `lib/infrastructure/storage/vault_path_strategy.dart`：vault 路徑慣例

# 其他主要流程

## Session Timeout

- app 進入背景後開始記錄離開前景時間
- 超過 timeout 回到前景時，會使當前 session 失效
- 若 trusted device 可用，嘗試重新還原 session
- 若 trusted state 失效，改為 `recoveryRequired`

## Index Database

- index database 只在解鎖 session 期間開啟
- 開啟時以 recovery wrapping key 導出的 key 做資料庫加密
- 若偵測到舊的 plaintext index，直接刪除重建

## 備份

### 本機備份

- 將 vault 內容封裝為 `.jbackup`
- 可透過檔案選擇器匯出

### Google Drive

- 上傳 `.jbackup` 到 Drive
- 可列出、下載並還原既有備份

## 還原

- 還原本機或 Drive 備份後
  - 清除現有解鎖狀態
  - 重設 repository / index 狀態
  - 重新跑 app startup 流程

## Markdown 匯出

- 僅在解鎖 session 有效時允許
- 匯出內容為解密後的 Markdown，不包含 vault 加密格式

# 個人化設定

這份文件整理 Quill Diary 目前的個人化設定行為，包括語言、主題、自動鎖定、圖片品質與日記排版。

它關心的是「可調哪些偏好、實際怎麼套用、存在哪裡、哪些條件下能不能改」，不重講 session 或加密全貌。

## 先講結論

- 個人化設定存在 vault 外的 `app_preferences.json`，不屬於正式日記資料，也不包含在 `backup_*.zip`。
- 設定頁沒有「儲存」按鈕，切換選項或調整滑桿後會立即寫回偏好檔。
- 預設值是：語言 `zh`、主題 `system`、自動鎖定 `3 分鐘`、圖片品質 `standard`、排版使用 `EditorTypographyPreferences.defaults`。
- 自動鎖定不是永遠都能改；目前實作要求「已有 recovery key 且 session 已解鎖」才可修改。
- 日記排版不只影響編輯器和預覽，也會影響首頁列表與摘要文字。

## 目前有哪些設定

| 類別 | 可調內容 | 預設值 | 備註 |
|------|----------|--------|------|
| 語言 | `繁體中文`、`English` | `繁體中文` | 直接切換 App locale |
| 自動鎖定 | `1 / 3 / 5 / 10 分鐘` | `3 分鐘` | 需已解鎖且已有 recovery key 才可修改 |
| 圖片品質 | `標準`、`高畫質`、`原圖` | `標準` | 只影響新加入編輯器的圖片 |
| 外觀主題 | `跟隨系統`、`淺色`、`深色` | `跟隨系統` | 直接對應 Flutter `ThemeMode` |
| 日記排版 | 標題字級、標題行距、內文字級、內文行距、段落間距 | 內建預設值 | 立即套用，可還原預設 |

## 設定怎麼套用

### 共同規則

- 設定頁使用 `personalizationPreferencesProvider` 載入與觀察目前偏好。
- 使用者一變更設定，`PersonalizationPreferencesController` 會立刻寫回 `UserPreferences.savePersonalizationPreferences(...)`。
- 載入失敗或尚未載入完成時，widget 層會暫時退回 `PersonalizationPreferences.defaults`。
- 未知或損壞的偏好值會回退到各自的預設值，而不是讓 App 進入不合法狀態。

### 語言

- 目前只支援 `zh` 與 `en`。
- `zh` 對應 `appZhLocale`，`en` 對應 `appEnLocale`。
- App 啟動時，`MaterialApp.router.locale` 直接讀這份偏好。
- 若使用者從未明確儲存語言偏好，預設就是 `zh`。
- 設定頁按鈕文字分別是 `繁體中文` 與 `English`。

### 外觀主題

- 主題偏好列舉為 `system / light / dark`。
- 會直接映射到 Flutter 的 `ThemeMode.system / light / dark`。
- App 啟動時，`MaterialApp.router.themeMode` 直接讀這份偏好。
- 這裡只改亮暗模式，不處理另外一套自訂色票系統。

### 自動鎖定

- 偏好值是 `SessionBackgroundTimeoutMinutes`，目前只有 `1 / 3 / 5 / 10` 分鐘四個選項。
- UI 會把「分鐘」單位獨立顯示在分段選項右側，不重複塞進每個按鈕。
- App 背景停留超過所選時間後，回到前景時需要重新驗證。
- `readSessionBackgroundTimeout()` 會把這份偏好轉成 `Duration` 供 session 模組讀取。
- 若設定頁當下沒有已解鎖 session，或尚未建立 recovery key，這個區塊會顯示鎖定訊息而不可修改。

補充：

- 文件與 UI 都描述成「背景放一段時間後才鎖」，不是每次切出 App 都立刻鎖。
- 備份、還原或匯入匯出等流程中的自動鎖定例外，請看 [../安全/解鎖與會話.md](../安全/解鎖與會話.md)。

### 圖片品質

這個設定只影響「從編輯器新選取、準備加入草稿或日記的圖片」。

| 選項 | 實際行為 |
|------|------|
| `標準` | 長邊縮至 `1280 px`、JPEG 品質 `70` |
| `高畫質` | 長邊縮至 `1920 px`、JPEG 品質 `85` |
| `原圖` | 不壓縮，不改解析度與檔案大小 |

目前實作補充：

- 預設值是 `standard`，不是 `original`。
- 被壓縮的圖片會轉成 JPEG。
- GIF 不會被轉 JPEG，即使選的不是原圖也會保留原樣。
- 它不會回頭重壓既有附件，只影響之後新加入編輯器的圖片。
- 它影響的是草稿暫存前的圖片準備流程，後續正式儲存進 `vault/` 的就是這份處理後結果。

### 日記排版

可調欄位共有五個：

- 標題字體大小
- 標題行距
- 內文字體大小
- 內文行距
- 內文段落間距

實際範圍如下：

| 欄位 | 範圍 | 預設值 |
|------|------|--------|
| 標題字體大小 | `18 ~ 28` | `22` |
| 標題行距 | `1.0 ~ 2.0` | `1.25` |
| 內文字體大小 | `14 ~ 22` | `16` |
| 內文行距 | `1.2 ~ 2.4` | `1.75` |
| 內文段落間距 | `0 ~ 32` | `8` |

套用範圍：

- 日記編輯器標題與內文
- Markdown 預覽
- 首頁列表標題
- 首頁列表摘要
- 首頁精簡摘要樣式

補充：

- `EditorTypographyPreferences.copyWith()` 與儲存流程都會自動 clamp，避免寫入超出範圍的值。
- 個人化頁有「還原預設」按鈕，會先跳確認對話框，再一次恢復五個排版欄位的預設值。
- 還原預設成功後會顯示底部通知。

## 設定資料放在哪裡

個人化設定不放進 `vault/`，而是獨立存放在 App support directory 下的：

- `app_preferences.json`

實際由 [`user_preferences.dart`](../../../lib/infrastructure/preferences/user_preferences.dart) 負責讀寫，路徑會組成：

- `getApplicationSupportDirectory()/AppIdentifiers.appStorageDirectory/app_preferences.json`

這代表它屬於 App 本機偏好，不是正式日記資料的一部分。

高層分工：

- `vault/`：正式日記、附件、復原設定與其他正式資料
- `drafts/`：編輯中的本地草稿與 pending 附件
- `app_preferences.json`：語言、主題、自動鎖定、圖片品質、日記排版

## `app_preferences.json` 目前保存哪些鍵

| key | 用途 |
|-----|------|
| `image_compress_preset` | 圖片品質偏好 |
| `session_background_timeout_minutes` | 自動鎖定分鐘數 |
| `theme_mode` | 主題模式 |
| `app_locale` | App 語言 |
| `editor_title_font_size` | 標題字級 |
| `editor_title_line_height` | 標題行距 |
| `editor_body_font_size` | 內文字級 |
| `editor_body_line_height` | 內文行距 |
| `editor_body_paragraph_spacing` | 內文段落間距 |

相容性補充：

- 舊資料若把自動鎖定存成 `30`，目前會被讀成 `10 分鐘`。
- 未知字串或空值會回退到各欄位預設值。

## 與其他模組的關係

| 設定 | 主要影響模組 |
|------|--------------|
| 語言 | `lib/app/app.dart`、整體 l10n UI |
| 外觀主題 | `MaterialApp.themeMode` 與整體畫面亮暗模式 |
| 自動鎖定 | session provider、背景逾時判斷、重新驗證流程 |
| 圖片品質 | 日記編輯器的圖片挑選、壓縮與草稿附件暫存 |
| 日記排版 | 編輯器、預覽、首頁列表與摘要 |

## 測試與驗證線索

- `test/smoke/app/app_test.dart`：確認 App 會套用儲存的 locale。
- `test/smoke/app/router_test.dart`：確認 router 可進入個人化頁。
- `test/application/session/session_providers_test.dart`：確認 session 讀取個人化設定中的背景逾時。
- 若之後調整選項數量、預設值或鍵名，至少要一併檢查 `personalization_providers.dart`、`user_preferences.dart`、`personalization_sections.dart` 與對應 l10n。
